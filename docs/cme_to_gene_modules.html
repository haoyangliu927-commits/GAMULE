CME 到 Gene Module 推断流程

这个包把流程拆成三段：scRNA-seq expression 计算 CME，CME/Jaccard 生成正负监督，最后用超边模型推断 gene modules。默认使用 6 个有意义超边，加 1 个无意义/未分配超边。

1. 输入与核心符号

对象含义代码位置
Xgene × cell 表达矩阵，从 h5ad 的 cell × gene 转置得到。src/cme_supervision.py
CME每个基因对的互斥分数。compute_cme_matrix
pval通过 gene 内 cell shuffle 得到的 high-tail permutation p-value。compute_cme_pvalues
negative_maskCME true，即互斥关系，作为负监督。build_supervision_from_cme
positive_maskJaccard 同类邻居及邻居的邻居，去掉负监督重叠后作为正监督。build_supervision_from_cme

2. CME 与假设检验

对每对基因 i, j，CME 使用共同表达最小值占比定义：

min_sum = sum_c min(X[i, c], X[j, c])
CME(i, j) = 1 - max(min_sum / sum_c X[i, c], min_sum / sum_c X[j, c])

p-value 使用置换零分布：对每个基因的表达在 cell 维度内独立 shuffle，重复 n_permutations 次，计算高尾概率：

p(i, j) = (#{CME_null(i, j) >= CME_observed(i, j)} + 1) / (n_permutations + 1)

互斥 true 的判定：

negative_mask = (CME > t_CME) & (pval < t_p)

3. Jaccard 正监督

每个基因的 CME true 行向量视为 CME vector。任意两个基因的 Jaccard Index 为：

Jaccard(i, j) = #[1,1] / (#[0,1] + #[1,0] + #[1,1])

实现上用一次矩阵乘法得到 #[1,1]：

intersection = CME_binary_float @ CME_binary_float.T
union = row_sum[:, None] + row_sum[None, :] - intersection
jaccard = intersection / union

然后按你的规则生成正监督：

  1. Jaccard > t_Jaccard 的基因对是潜在同类。
  2. 对每个基因只保留 top k_Jaccard 个最高 Jaccard 同类邻居；如果不够 k 个就保留已有的，不增补。
  3. 对基因 g，取直接同类邻居 N_g 和邻居的邻居 N_g_g 作为正监督候选。
  4. 最终正监督会去掉所有负监督重叠，避免一个基因对同时被正负约束。

4. 超边模型与未分配超边

默认 num_gene_modules = 6，num_hyperedges = 7。前 6 个超边参与训练和相似度计算；第 7 个只接收完全没有出现在任何监督矩阵里的基因。

关键设计：第 7 个无意义超边不进入训练时的 partition @ hyperedge_emb，最终 gene_emb 也只由前 6 个有意义超边生成。因此未分配超边不会污染有意义 gene modules。

expression = load_expression_from_h5ad("datasets/adata_672.h5ad", num_genes=420)
cme_matrix = compute_cme_matrix(expression.matrix)
pval_matrix = compute_cme_pvalues(expression.matrix, cme_matrix, n_permutations=20)

supervision = build_supervision_from_cme(
    cme_matrix,
    pval_matrix,
    cme_threshold=0.9,
    pvalue_threshold=0.05,
    jaccard_threshold=0.7,
    top_k=10,
)

result = run_supervised_hyperedges(
    pos_mask=supervision.positive_mask,
    neg_mask=supervision.negative_mask,
    num_hyperedges=7,
    use_unassigned_hyperedge=True,
)
